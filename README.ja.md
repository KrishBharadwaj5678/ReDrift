<h1 align="center"><img src="https://github.com/KrishBharadwaj5678/ReDrift/raw/main/favicon/favicon.svg" width="35"/> ReDrift</h1>

<p align="center">
  ReDrift は、輝くライトトレイルとダイナミックなモーションエフェクトの中を果てしなく進み続ける旅をシミュレートした、没入感あふれる 3D ビジュアル体験です。
</p>

![ReDriftDemo](https://github.com/KrishBharadwaj5678/ReDrift/raw/main/ReDriftDemo.png)

## <img src="https://github.com/KrishBharadwaj5678/ReDrift/raw/main/assets/readme/features.gif" width="35"/> 主な機能

| 機能 | 説明 |
|------|------|
| 🌌 抽象的なビジュアル | 光り輝くオブジェクトが絶え間なく前方へ進む幻想的な演出 |
| 🎮 インタラクティブな 3D シーン | Three.js と WebGL によるリアルタイムレンダリング |
| ⚡ スムーズなパフォーマンス | 滑らかな映像体験のために最適化されたアニメーション |
| 🌐 ブラウザで動作 | インストール不要でブラウザからすぐに実行可能 |
| 🎨 ミニマルな UI | ビジュアル体験に集中できるシンプルなインターフェース |

## <img src="https://github.com/KrishBharadwaj5678/ReDrift/raw/main/assets/readme/gettingStarted.gif" width="35"/> はじめに

### 1️⃣ リポジトリをクローンする

```bash
git clone https://github.com/KrishBharadwaj5678/ReDrift.git
```

### 2️⃣ プロジェクトディレクトリへ移動する

```bash
cd ReDrift
```

### 3️⃣ ブラウザで開く

`index.html` をブラウザで開くだけで実行できます。
