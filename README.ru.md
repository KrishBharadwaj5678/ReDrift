<h1 align="center"><img src="https://github.com/KrishBharadwaj5678/ReDrift/raw/main/favicon/favicon.svg" width="35"/> ReDrift</h1>

<p align="center">
  ReDrift это захватывающий трёхмерный визуальный проект, который имитирует бесконечное путешествие сквозь светящиеся световые трассы и динамические эффекты движения.
</p>

![ReDriftDemo](https://github.com/KrishBharadwaj5678/ReDrift/raw/main/ReDriftDemo.png)

## <img src="https://github.com/KrishBharadwaj5678/ReDrift/raw/main/assets/readme/features.gif" width="35"/> Возможности

| Возможность | Описание |
| ------------ | ------------------------------------------------------------ |
| 🌌 Абстрактная визуализация | Завораживающие светящиеся элементы с непрерывным движением вперёд |
| 🎮 Интерактивная 3D-сцена | Рендеринг в реальном времени с использованием WebGL и Three.js |
| ⚡ Плавная производительность | Оптимизированная анимация для максимально плавного визуального опыта |
| 🌐 Работает в браузере | Установка не требуется запускается прямо в браузере |
| 🎨 Минималистичный интерфейс | Полное внимание сосредоточено на визуальном путешествии |

## <img src="https://github.com/KrishBharadwaj5678/ReDrift/raw/main/assets/readme/gettingStarted.gif" width="35"/> Начало работы

### 1️⃣ Клонируйте репозиторий

```bash
git clone https://github.com/KrishBharadwaj5678/ReDrift.git
```

### 2️⃣ Перейдите в каталог проекта

```bash
cd ReDrift
```

### 3️⃣ Откройте в браузере

Просто откройте файл `index.html` в любом современном браузере.
