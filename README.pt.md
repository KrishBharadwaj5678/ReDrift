<h1 align="center"><img src="https://github.com/KrishBharadwaj5678/ReDrift/raw/main/favicon/favicon.svg" width="35"/> ReDrift</h1>

<p align="center">
  ReDrift é uma experiência visual 3D imersiva que simula uma jornada infinita através de trilhas luminosas e efeitos dinâmicos de movimento.
</p>

![ReDriftDemo](https://github.com/KrishBharadwaj5678/ReDrift/raw/main/ReDriftDemo.png)

## <img src="https://github.com/KrishBharadwaj5678/ReDrift/raw/main/assets/readme/features.gif" width="35"/> Recursos

| Recurso | Descrição |
|---------|-----------|
| 🌌 Visuais Abstratos | Elementos luminosos fascinantes em movimento contínuo para a frente |
| 🎮 Cena 3D Interativa | Renderização em tempo real utilizando WebGL com Three.js |
| ⚡ Alto Desempenho | Animações otimizadas para uma experiência visual fluida |
| 🌐 Baseado no Navegador | Nenhuma instalação necessária — funciona diretamente no navegador |
| 🎨 Interface Minimalista | Foco total na experiência visual |
